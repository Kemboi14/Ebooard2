File Guidelines & Security

Supported File Types

  • • PDF documents (.pdf)
  • • Word documents (.doc, .docx)
  • • Excel spreadsheets (.xls, .xlsx)
  • • PowerPoint presentations (.ppt, .pptx)
  • • Text files (.txt, .rtf)
  • • Images (.jpg, .jpeg, .png, .gif, .bmp, .tiff, .webp)
  • • Archives (.zip, .rar, .7z)
  • • Data files (.json, .xml, .csv, .html)

Security Features

  • • Automatic file type detection
  • • Malware scanning integration
  • • Content validation
  • • Secure file storage
  • • Access control enforcement
  • • Audit trail logging

Document Status

  • • Draft: Internal document, not yet ready for publication
  • • Under Review: Document is being reviewed
  • • Approved: Document approved but not yet published
  • • Published: Document is live and accessible
  • • Archived: Document is no longer active
